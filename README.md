# e-Shift Household Goods Shifting Management System

## Project Overview
The e-Shift Management System is a Windows-based software solution developed to automate and manage the daily operational activities of a household goods shifting company. The system supports both **Customer** and **Admin** roles, ensuring smooth coordination of transport jobs, load handling, and resource allocation.

The application improves operational efficiency by maintaining accurate records, tracking transport activities, and providing management insights for better decision-making.

## Key Features
### Customer Side
- Customer registration and profile management
- Transport job requests
- Load creation and tracking
- Pickup and delivery location management
- View job status and service history

### Admin Side
- Customer, job, and load management
- Allocation of transport units (lorry, driver, assistant, container)
- Job and load tracking
- Monitoring pickup and delivery progress
- Report generation for management insights

## System Highlights
- Unique identification for customers, jobs, and loads
- Accurate and centralized record keeping
- Improved resource utilization
- Supports operational efficiency and service quality

## Technology Stack
- **Programming Language:** C#  
- **IDE:** Visual Studio  
- **Database:** MySQL  
- **Design Tools:** Draw.io  
- **Version Control:** GitHub  

## System Architecture
- Windows-based desktop application developed using C#
- MySQL database for data storage and management
- Role-based access for Customer and Admin users
- Logical system flow and diagrams designed using Draw.io

## Installation & Setup
1. Clone the repository from GitHub.
2. Open the solution file (`.sln`) using Visual Studio.
3. Configure the MySQL database and import the provided schema.
4. Update database connection settings in the application.
5. Build and run the application from Visual Studio.
